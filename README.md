# Метод Гаусса в кольце вычетов

> Курсовая работа по дисциплине «Алгоритмы и структуры данных»

## Описание

Реализация решения систем линейных уравнений (СЛУ) методом Гаусса над **кольцом вычетов по модулю m** на языке C++.

Программа выполняет прямой и обратный ход метода Гаусса, автоматически определяет тип решения системы и выводит результат в одном из трёх видов:
- единственное решение,
- бесконечное множество решений (фундаментальная система решений),
- отсутствие решений.

## Особенности

- Все арифметические операции выполняются **по модулю m** в кольце вычетов Z/mZ
- Поиск обратного элемента по модулю методом перебора
- Поддержка матриц произвольного размера (не только квадратных)
- Ввод матрицы из текстового файла или генерация случайной матрицы
- Вычисление ранга матрицы коэффициентов и расширенной матрицы для определения типа решения
- Перемещение нулевых столбцов вправо с сохранением маппинга переменных
- Измерение времени выполнения (закомментировано, можно включить)

## Использование

При запуске программа запрашивает три параметра:

```
Enter the number of equations: <количество уравнений>
Enter the number of variables: <количество переменных>
Enter the module: <модуль m>
```

Расширенная матрица СЛУ (коэффициенты + столбец свободных членов) считывается из файла `TextFile1.txt`, расположенного в рабочей директории. Элементы записываются построчно, разделённые пробелами или переносами строк.

**Пример `TextFile1.txt`** для системы 2×2 по модулю 7:

```
1 2 3
4 5 6
```

## Описание алгоритма

| Шаг | Действие |
|-----|----------|
| 1 | Прямой ход: приведение матрицы к ступенчатому виду с перестановками строк |
| 2 | Вычисление рангов матрицы коэффициентов и расширенной матрицы |
| 3 | Определение типа решения: нет решений / единственное / бесконечно много |
| 4 | Перемещение нулевых столбцов вправо (для случая бесконечного решения) |
| 5 | Обратный ход: приведение к диагональному виду |
| 6 | Нормировка строк (умножение на обратный элемент по модулю) |
| 7 | Вывод решения |

**Временная сложность:** O(n² · m) для прямого и обратного хода, где n — размер системы, m — модуль (из-за поиска обратного элемента перебором).

## Класс `Matrix`

Основная логика сосредоточена в классе `Matrix`:

- `gaussEliminationForward()` — прямой ход метода Гаусса
- `gaussEliminationBackward()` — обратный ход
- `solve()` — полное решение СЛУ
- `calculateRank()` — вычисление ранга
- `modularInverse()` — нахождение обратного элемента по модулю
- `generateFundamentalSystem()` — формирование фундаментальной системы решений
- `add()`, `subtract()`, `multiply()` — арифметика матриц по модулю
- `readMatrixFromFile()` — чтение матрицы из файла
